**ВАРІАНТ - 6**

MonoBehaviour — це базовий клас у Unity, від якого успадковуються скрипти для створення компонентів. 
Він надає інтерфейс для взаємодії з ігровим рушієм через методи життєвого циклу та доступ до властивостей об’єкта.

**1. Ключові властивості**    
enabled: Контролює активність компонента. Якщо false, методи оновлення (Update) не викликаються.  
gameObject: Посилання на об'єкт-контейнер, до якого прикріплено скрипт.  
transform: Швидкий доступ до просторових характеристик об'єкта (позиція, поворот, масштаб).    
  
**3. Методи життєвого циклу**    
Start(): Ініціалізація логіки перед першим кадром.  
Update(): Виконання щокадрових операцій (ввід, таймери).  
FixedUpdate(): Робота з фізикою (виконується з константним кроком часу).  
LateUpdate(): Логіка після всіх оновлень (корекція камери, стеження).
